# Assignment: Session 3

**Submission Guidelines:** Submit a ZIP file of your Django project folder (excluding the virtual environment folder).

### The "Personal Blog" Project - Part 1
We are going to start building a personal blog that we will add to over the next few weeks.

**Requirements:**
1. Create a project called `my_blog`.
2. Create an app called `posts`.
3. Register the app.
4. Create a view called `all_posts` that returns an `HttpResponse` saying `"Welcome to my awesome blog!"`.
5. Route this view to the URL: `http://127.0.0.1:8000/posts/`.

**Deliverable:**
Zip your `my_blog` folder and submit it. **Important:** Do NOT zip your virtual environment folder (`venv` or `djenv`), as it is very large and not needed to run the code. Just zip the project folder that contains `manage.py`.
